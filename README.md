# MyL CLI

AI-powered command-line interface for secure digital identity management.

## Overview

MyL CLI is a terminal-based AI assistant that connects to the MyL.zip platform for:
- SSL certificate management
- API key generation and rotation
- Security monitoring and analytics
- Intelligent automation via OpenRouter AI models
