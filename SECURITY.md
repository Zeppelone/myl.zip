# Security Policy

If you discover a security vulnerability, please report it privately via GitHub Security Advisories:

https://github.com/Zeppelone/myl.zip/security/advisories/new

Do not open a public issue.

## Supported Versions
Currently under active development (pre-1.0). Security fixes will target `main`.

## Guidelines
- Never include secrets or personal data in issues/PRs.
- Provide minimal reproduction steps.
- Share environment details when safe.
